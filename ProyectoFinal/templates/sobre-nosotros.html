Sobre nosotros

Somos estudiantes de la 2da etapa ''Desarrollo Web'' del Informatorio. Conformamos un equipo de desarrollo de 4 integrantes, trabajando en conjunto y de manera remota

Sobre el proyecto

Este es un proyecto de tipo 'blog' creado enteramente por el grupo.Y cuenta con las siguientes caracteristicas:

  • Construido en el framework Django , utilizando Python como lenguaje
  • Conectado a una base de datos sqlite3 que nos provee Django, para persistir registros de usuarios,fotos,noticias,etc
  • Para las vistas y lo visual se utilizaron templates, asi como Bootstrap y Css puro para los estilos
  • Fue desarrollado en el patron de arquitectura 'mvc', en django aplicado como 'mvt' Modelo Vista Template
