Sobre nosotros

Somos estudiantes de la 2da etapa ''Desarrollo Web'' del Informatorio. Conformamos un equipo de desarrollo de 5 integrantes, trabajando en conjunto y de manera remota.

Sobre el proyecto

Este es un proyecto de tipo 'blog' creado enteramente por el grupo, el cual cuenta con las siguientes caracteristicas:

  • Construido con el framework Django, utilizando Python como lenguaje de programación principal.
  • Conectado a una base de datos sqlite3, para persistir registros de usuarios, fotos, noticias, etc.
  • Para la interfaz visual utlizamos el lenguaje de marcado HTML para crear los templates y para agregarle estilos el framework de Boostrap, que utiliza el método de las clases.
  • Fue desarrollado en el patron de arquitectura 'MVC', en django aplicado como 'MVT', Modelo Vista Template.
